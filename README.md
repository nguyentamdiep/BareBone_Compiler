# Bare Bone Programming Language

| Barebone command        | explain        |
|-------------------------|----------------|
| init X 5                | X:=5           |
| in X                    | cin>>X         |
| out X                   | cout<<X        |
| clear X                 | X:=0           |
| incr X                  | X:=X+1         |
| decr X                  | X:=X-1         |
|while X not 0 do         |while X not 0 do|       
|   ...                   |  decr X        |        
|end	                    |end	  	       |
